Notifier

Descripción

La clase Notifier gestiona y centraliza las notificaciones del sistema utilizando sesiones PHP.

Permite generar mensajes de éxito, error, advertencia e información, almacenarlos por método de presentación y mostrarlos posteriormente mediante Bootstrap Alerts, Toasts o SweetAlert.

Incluye métodos de validación para comprobar si existen notificaciones pendientes o si existe un tipo específico antes de ejecutar lógica condicional en la aplicación.

Ejemplos de uso

1. Crear una notificación básica

(new Notifier())
  ->message('Operación realizada correctamente')
  ->add();

2. Notificación de éxito

(new Notifier())
  ->success('Datos guardados correctamente')
  ->add();

3. Notificación de error (danger)

(new Notifier())
  ->danger('Ocurrió un error')
  ->add();

4. Notificación de advertencia

(new Notifier())
  ->warning('Advertencia importante')
  ->add();

5. Notificación informativa

(new Notifier())
  ->info('Información del sistema')
  ->add();

6. Notificación usando Bootstrap (por defecto)

(new Notifier())
  ->success('Mensaje Bootstrap')
  ->bootstrap()
  ->add();

7. Notificación tipo Toast

(new Notifier())
  ->info('Mensaje tipo Toast')
  ->toast()
  ->add();

8. Notificación con SweetAlert

(new Notifier())
  ->danger('Mensaje SweetAlert')
  ->sweetalert()
  ->add();

9. Agregar múltiples notificaciones

$notifier = new Notifier();

$notifier->success('Registro creado')->add();
$notifier->info('Bienvenido')->add();
$notifier->warning('Perfil incompleto')->add();

10. Verificar si existen notificaciones

$notifier = new Notifier();

if ($notifier->any()) {
  echo 'Hay notificaciones pendientes';
}

11. Verificar si existe un tipo específico

$notifier = new Notifier();

if ($notifier->has('danger')) {
  echo 'Existen errores';
}

12. Uso del método can()

El método can() se utiliza para validar la existencia de notificaciones antes de ejecutar acciones específicas. 

$notifier = new Notifier();

if ($notifier->can()->any()) {
  echo 'Hay mensajes disponibles';
}

13. Uso de can() con validación de tipo

$notifier = new Notifier();

if ($notifier->can()->has('danger')) {
  echo 'Hay errores registrados';
}

14. Uso real con condición negativa (caso típico)

$notifier = new Notifier();

if (!$notifier->can()->has('danger')) {
  // Continuar flujo normal
}

15. Ejemplo solicitado: can() + danger()

$notifier = new Notifier();

if (!$notifier->can()->danger()) {
  // No hay mensajes de error
}

16. Mostrar notificaciones Bootstrap

(new Notifier())->showBootstrap();

17. Mostrar notificaciones Toast

(new Notifier())->showToasts();

18. Mostrar notificaciones SweetAlert

(new Notifier())->showSweetAlerts();

19. Flujo completo recomendado

$notifier = new Notifier();

$notifier->success('Operación correcta')->add();
$notifier->warning('Advertencia')->toast()->add();

if (!$notifier->can()->has('danger')) {
  $notifier->showBootstrap();
  $notifier->showToasts();
}
