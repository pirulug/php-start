TextTransformer

Descripción

La clase TextTransformer permite transformar y normalizar textos mediante una interfaz fluent (encadenamiento de métodos).

Soporta reglas de reemplazo basadas en expresiones regulares, presets predefinidos reutilizables, transformación de mayúsculas y minúsculas, transliteración de caracteres Unicode y generación de identificadores cortos.

Está pensada para casos como generación de slugs, usernames seguros, normalización de texto, IDs cortos y transformaciones personalizadas.

Ejemplos de uso

1. Uso básico

$text = (new TextTransformer())
  ->text('Hola Mundo')
  ->apply();

2. Reemplazo personalizado con expresión regular

$text = (new TextTransformer())
  ->text('Hola Mundo')
  ->replace('/\s+/', '_')
  ->apply();

3. Convertir texto a minúsculas

$text = (new TextTransformer())
  ->text('Hola Mundo')
  ->lowercase()
  ->apply();

4. Convertir texto a mayúsculas

$text = (new TextTransformer())
  ->text('Hola Mundo')
  ->uppercase()
  ->apply();

5. Aplicar transliteración (acentos y caracteres especiales)

$text = (new TextTransformer())
  ->text('Canción número uno')
  ->transliterate()
  ->apply();

6. Generar un slug usando preset

$slug = (new TextTransformer())
  ->text('Título del Artículo')
  ->slug()
  ->lowercase()
  ->apply();

7. Generar un username seguro

$username = (new TextTransformer())
  ->text('Usuario Demo 123')
  ->username()
  ->lowercase()
  ->apply();

8. Aplicar preset leet

$text = (new TextTransformer())
  ->text('Password')
  ->leet()
  ->apply();

9. Aplicar preset safe

$text = (new TextTransformer())
  ->text('Texto con símbolos !@#$')
  ->safe()
  ->apply();

10. Encadenar múltiples presets

$text = (new TextTransformer())
  ->text('Texto de Ejemplo!')
  ->slug()
  ->leet()
  ->apply();

11. Generar un identificador corto (shortId)

$id = (new TextTransformer())
  ->shortId()
  ->apply();

12. Generar shortId con longitud personalizada

$id = (new TextTransformer())
  ->shortId(12)
  ->apply();

13. Generar shortId con alfabeto personalizado

$id = (new TextTransformer())
  ->shortId(6, 'abcdef0123456789')
  ->apply();

14. Resetear reglas personalizadas

$text = (new TextTransformer())
  ->text('Texto Inicial')
  ->replace('/\s+/', '-')
  ->reset()
  ->apply();

15. Usar presets personalizados por configuración

$transformer = new TextTransformer([
  'custom' => [
    '/\s+/' => '.',
    '/[^a-z0-9\.]/i' => ''
  ]
]);

$result = $transformer
  ->text('Texto Personalizado!')
  ->custom()
  ->apply();

Presets disponibles por defecto

  • slug
  • username
  • leet
  • safe

Notas de funcionamiento

  • Las reglas se aplican en el orden en que fueron definidas
  • El método apply() devuelve siempre un string
  • Los presets se pueden invocar como métodos dinámicos
  • La transliteración requiere la extensión intl
  • Los caracteres finales '-' y '_' se eliminan automáticamente
